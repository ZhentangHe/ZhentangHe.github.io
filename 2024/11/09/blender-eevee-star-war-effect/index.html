Blender EEVEE 星球大战效果

Zhentang He
2024-11-09
建模与渲染
Blender, EEVEE, Visual Effect

管道实现

  1. Shift a 添加一个柱体， r x 90 Enter 以 x 轴为轴旋转 90 度， s y 8 沿 y 轴缩放 8 倍。
  2. Tab 进入编辑模式，在左上角选择面选择模式，按住 Shift 选择柱体的两个面，x 删除这两个面。

粒子生成

  1. 再次 Tab 进入物体模式，Shift a 添加一个经纬球，g 将它挪开，s y 沿 y 轴放大到合适大小，s 调整缩小到合适大小。
  2. 按住 z 键转入渲染模式，在材质选项卡中新建一个材质，将表面类型从 Principled BSDF/ 原理化 BSDF 改为 Emission/ 自发光，调整合适的颜色和强度/力度。
  3. 按住 z 回到实体模式，选中柱体，在粒子选项卡中新建一个粒子系统，将类型从 Emitter/ 发射器 改为 Hair/ 头发，在渲染属性中将 Render As/ 渲染为 从 Path/ 路径 改为 Object/ 对象，在 Instance Object/ 实例对象 中选择球体，勾选 Object Rotation/ 物体旋转，调整 Scale/ 缩放 和 Scale Randomness/ 缩放随机性 到合适大小。
  4. 按住 z 键转入渲染模式，把世界选项卡中的背景颜色改为黑色以便于观察效果。
  5. （可选）选中柱体，在修改器选项卡中添加一个 Wireframe/ 线框 修改器，调整 Thickness/ 粗细 到0。
  6. 选中柱体，m添加一个新的集合，将柱体移动到集合中，Shift a 添加一个复制，Shift Space + g 切换成移动模式，按住 Ctrl 拖动坐标轴复制到合适位置，Shift d 复制多次，并使用鼠标中键限制移动方向。

摄像机运动

  1. ~切换到前视图，Shift a 添加一个摄像机，此时摄像机的视角刚好位于柱体构成的管道中并望向前方。使用 Shift Space + g 切换成移动模式，按住 Ctrl 拖动坐标轴移动到管道的一端，如果有其他摄像机就删除掉，小键盘 0 切换到摄像机视角检查效果。
  2. 选中摄像机，在（默认）第一帧给它的 y 轴位置添加一个关键帧，然后在下方的时间轴中选择 80 帧，将摄像机移动到管道的另一端，再次给它的 y 轴位置添加一个关键帧，按 Space 播放动画查看效果。
  3. 选中绿色的摄像机图标，将 Focal Length/ 焦距 调整到合适大小，按 Space 播放动画查看效果。
  4. 此外，可以调整球体大小和粒子数量等改善效果。
  5. Shift a 添加一个立方体，s 8 放缩并移入之前创建的集合中，切换到上方的着色选项卡，新建一个材质，删除 Principled BSDF/ 原理化 BSDF，添加 Principal Volume/ 原理化体积，把体积属性连接到材质输出的体积上，适当降低 Density/ 密度。

关于 Bloom 效果

可以在 Render Properties/ 渲染属性 中找到 Bloom/ 泛光 选项，但是在 EEVEE 中已经被移除了。因此我们可以使用合成器来模拟泛光效果。
打开合成选项卡，拉出来一个新的视图用来实时查看效果（记得把视图着色方式的合成器从已禁用改为摄像机或者总是），在合成视图中勾选 Use Nodes/ 使用节点，Shift a 添加 Glare/ 辉光 节点，挪到两个节点之间自动连接，把条斑改成辉光，调整其他属性来模拟泛光效果。
