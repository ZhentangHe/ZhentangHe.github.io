前言

本文使用 Steam 版本的 最新 LTS Blender，4.x 版本的 Blender 带来了很多新特性，比如新的渲染引擎 Cycles X，新的建模工具等等，同时也有一些改动，比如移除了 EEVEE 的 Bloom 效果，调整了一些快捷键等等。主要是为了记录一些常用的操作和技巧，方便本人日后查阅。

建模 编辑模型
雕刻 雕刻模型 ZBrush
UV编辑 模型 UV 展开
纹理绘制 贴图
着色 图形学 Shader
动画 动画制作

基础操作

比较方便的一点是 Blender 在使用组合键时，在最下方会显示当前的操作提示，这样可以方便查看快捷键。

视图操作

快捷键 作用
鼠标中键 旋转视图
Shift + 鼠标中键 平移视图
Ctrl + 鼠标中键 或 滚轮 缩放视图
Home 重置视图
小键盘 0 摄像机视图
小键盘 1 正视图
小键盘 3 右视图
小键盘 5 透视视图
小键盘 7 顶视图
小键盘 9 当前视图反转
左键拉动边框 新建并列窗口

布局操作

快捷键 作用
Shift + A 添加物体
T 显示/隐藏工具栏
Shift + Space 使用工具栏
框选状态下左键 选中物体
移动状态下左键 按轴/面移动物体
旋转状态下左键 按视线/面旋转物体
缩放状态下左键 按轴/面/整体缩放物体
左键长按缩放 缩放罩体
变换状态下左键 综合变换物体
以上状态下中键 吸附到轴上完成变换
G [X, Y, Z] 数字 沿轴移动物体
R [X, Y, Z] 数字 沿轴旋转物体
S [X, Y, Z] 数字 沿轴缩放物体
标注状态下左键 添加标注
测量状态下左键 测量距离
添加状态下左键 添加物体，会吸附到合适位置
N 显示/隐藏属性栏
Shift + D 复制物体

编辑模式

该模式下可以对物体进行编辑，包括顶点、边、面及其组合等。

快捷键 作用
Tab 进入/退出编辑模式（以上操作亦适用）
Alt + 左键 选择子元素/循环选择（顶点、边、面）
Alt + Ctrl + 左键 选择子元素/环形选择（边、面）
框选状态下E 挤出顶点/边/面，类似 SolidWorks 的拉伸
框选状态下Alt + E 其他挤出操作，默认选取的平均法向量，可选各朝向法向挤出
框选状态下I 内插面，组合挤出使用类似 SolidWorks 的抽壳
框选状态下Ctrl + B 倒角
框选状态下Ctrl + R 环切
框选状态下K 切割
框选状态下F 连接顶点线
框选状态下Ctrl + X 融并
框选状态下Ctrl + Shift + B 布尔操作

游标相关

快捷键 作用
Shift + 右键 移动游标
Shift + S 设置游标位置
Shift + C 重置游标位置
N + 锁定到3D游标 以游标为中心视图
